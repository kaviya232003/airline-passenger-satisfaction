# ✈️ Airline Passenger Satisfaction Dashboard

An interactive Tableau dashboard analyzing airline passenger satisfaction across travel classes, age groups, and service parameters.

---

## 📌 Project Overview

This project visualizes airline passenger satisfaction data to uncover patterns across economy and business class travelers, age demographics, and key service ratings. Built using Tableau Public with multiple interactive sheets combined into a final dashboard.

## 📈 Dashboard Sheets

| Sheet | Description |
|---|---|
| F/M Passenger | Gender-wise passenger breakdown |
| Total Passenger | Overall passenger count analysis |
| Economy Class | Satisfaction rate in economy class |
| 17 Age Passenger | Report of passengers below age 17 |
| Returning | Returning vs first-time passenger comparison |
| Class Passenger | Stratification rate by class (Business, Economy, Economy Plus) |
| Female Passenger | Female passenger satisfaction |
| Male Passenger | Male passenger satisfaction |
| Date | Time-based passenger trends |
| Ratings | Service parameter ratings (Wifi, Online Booking, Gate Location) |
| Final Dashboard | Combined interactive dashboard |

---

## 💡 Key Findings

- **81.69%** of passengers are returning travelers — strong customer loyalty
- Economy class shows **81.23% neutral or dissatisfied** — major area for improvement
- Passengers below age 17 show **83.27% neutral or dissatisfied**
- Economy class satisfaction is significantly lower than Business class
- **Gate Location** scores the highest rating at **2.98 / 3.00**
- Only **18.77%** of economy class passengers are satisfied
- Only **16.73%** of passengers below age 17 are satisfied
